# Coffee Kode

Coffee Kode comercializa de manera directa el café en grano y molido en todo el país, desde su cosecha en cafetaleras mexicanas hasta la entrega en miles de hogares, a un precio competitivo, que genera derrama económica para el sector campesino y ahorro significativo para el cliente.

## Proyecto

Diseño web de tienda en línea (e-commerce) de café en grano y molido para la venta a clientes desarrollado con React / JavaScript y Java / SpringBoot con MySQL bajo el marco de trabajo SCRUM.
